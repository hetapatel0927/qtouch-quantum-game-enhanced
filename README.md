# Quantum Touchdown – Enhanced Game Version

## Overview
This project is an enhanced version of the Quantum Touchdown Java game. It extends the original implementation by introducing dynamic gameplay configuration and new game mechanics, improving user interaction and overall game logic.

## Features
- Interactive GUI built using Java Swing  
- Dynamic input dialog to configure number of each card type  
- Game repetition based on user-defined card count  
- Special **"Mr. P" card** that instantly wins the game when selected  
- Event-driven gameplay logic and real-time updates  
- Runnable JAR file and batch script for easy execution  

## Technologies Used
- Java (Swing)  
- Object-Oriented Programming (OOP)  
- Event-Driven Programming  

## Project Structure
```
src/
 ├── qtouch/    (Game logic, GUI, and controllers)
```

## How to Run

### Option 1: Using Batch File (Recommended)
